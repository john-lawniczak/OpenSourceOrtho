# Example prompts

Reusable, copy-paste prompts for driving an AI coding/analysis agent against the
kinds of records OpenSource Ortho works with (intraoral STL scans, scanner-export
Rx files, intraoral photos).

These are **educational examples**, not clinical instructions. OpenSource Ortho is
not a medical device, not a diagnosis tool, and not an autonomous treatment
planner. Every prompt here is written to ask for an *independent sanity check with
its limitations stated* — never a diagnosis or a recommendation to undergo (or
skip) treatment. Decisions stay with a licensed professional and the patient.

| Prompt | Use case |
|--------|----------|
| [`transverse-arch-width-sanity-check.md`](transverse-arch-width-sanity-check.md) | Independent check of upper-vs-lower arch width from an iTero/OrthoCAD export, before agreeing to a palatal expander. |
